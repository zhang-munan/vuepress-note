# 介绍
可以同时兼容vue2、vue3的uni-app模板，封装了大量常用的共同方法，可以帮助提升开发效率

# 特征
- 基于 Promise 对象实现更简单的 request 使用方式[luch-request](https://www.quanzhan.co/luch-request/)，支持请求和响应拦截
- 精致的上拉刷新和上拉加载js框架[mescroll](http://www.mescroll.com/)
